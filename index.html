Projects

Here are some of the more notable projects I have worked on. Please click on the icons to see more about that project.

There are icons for github, heroku, site previews and an image to help you see what the projects look like.


  • Macro App "Mac Attack"

    This is an app that was created in Android Studio using Java. A user can log into their Dropbox via the app, and can send macros

    from the app to any computer that is logged into their Dropbox. It uses a combination of Dropbox API along with Autohotkey to accomplish this.

  • PHP-Based Storefront

    This was a page I did for school, but exercised a lot of creativity on. We were supposed to design a page with loose directions using PHP.

    It uses a live database (PostgreSQL) via Heroku and has a fully working login/registration system.

  • Javascript Soundboard

    While this was a school assignment, it's something I'm very proud of. We were asked to use Javascript events, and a few other topics.

    This could be done with multiple websites or with one using a combination of the, for a total of 13 topics. I made this soundboard completely from my own ideas without directions.

    It uses royalty-free piano sounds and a looping system I came up with to allow users to make a playlist out of the soundboard itself.

  • Studio Ghibli Fan Site

    I made this site as a final project for my web development class, but there were no other guidelines besides using an API and somehow making use of CSS.

    I'm proud of the styling and minimalist style that was used to present Studio Ghibli's API in an aesthetic way. I am a huge fan of Studio Ghibli and wanted to make this tribute.

  • Summoner's RPG "SRPG"

    This is an on-going project using RPG Maker and Javascript that I am working on with my younger brother. It's an ambitious open-world game that is a fusion of monster-hunting/taming and a card-game.

    We are continuing to work on it, but this game has pushed me to learn creative means and solutions for problems I come across.
